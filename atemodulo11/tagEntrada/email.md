# email
<input type="email">

- Espera que o usário digite um email
- Irá validar se o valor é um email

# Atributos

- placeholder
- readonly/disabled
- value

- required

- multiple
    * o campo irá receber 1 ou mais email, separado por vírgulas

- minlenght / maxlength
    * O mínimo e/ou máximo valor que o campo irá conter

- size
